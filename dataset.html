Dataset Access Policy
The dataset will be made available only to registered participants of the shared task.

Dataset Overview

The dataset is designed for research on Natural Language to Structured Text and Rust Code generation in industrial automation scenarios.

The corpus contains natural language descriptions of industrial control logic and their corresponding IEC 61131-3 Structured Text (ST) and Rust Code implementations.

Dataset Components

  • Natural Language Descriptions
  • Structured Text Programs
  • Rust Programs
